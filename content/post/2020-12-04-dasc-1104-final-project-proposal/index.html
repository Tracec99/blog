2.1 Data

I will be analyzing global agricultural development using data taken from the TidyTuesday GitHub page. The primary data I will be using is contained key_crop_yields.csv, which has 13,075 observations and 14 variables: The variable Entity, a character that denotes the name of the region or country the observation comes from, the variable Code, which is the country code of the country the observation comes from (continents and regions use NA, and Year, a double ranging from 1961-2014 that represents the year the observation comes from. Additionally, there are 11 other variables representing 11 different crops and their yield in metric tonnes per hectare. The crops represented are wheat, rice, maize, soybeans, potatoes, beans, peas, cassava, barley, cocoa beans, and bananas.

The other data of interest are found in arable_land_pin.csv, which contains the additional double variable Arable land needed to produce a fixed quantity of crops ((1.0 = 1961)), which can be used to track changes in land use efficiency over time. I will also be using the data in the files cereal_crop_yield_vs_fertilizer_application.csv and cereal_yields_vs_tractor_inputs_in_agriculture.csv, both of which track the use of fertilizer and tractors in agriculture over time, two important factors in farming efficiency. Fertilizer contains the additional variables Cereal yield and Nitrogen fertilizer use, while Tractor_inputs contains Tractors per 100 sq km arable land, Cereal yield (kilograms per hectare), and Total population, which is taken from the Gapminder dataset. Each of these 3 datasets also contain the key variables Entity, Code, and Year.

2.2 Questions

  • Question 1: What is the most abundant cereal crop that is grown in the world and has this changed since 1961? Is this popularity due to widespread use of the crop as a staple, or because of its use by specific population centers? This question can be answered by generating a line plot of the cereal yield of each crop across time. I will also pair the line plot with some sort of proportional plot of the most populous countries (taken from the gapminder variable in the tractor data) and the proportion of their total crop yield each cereal crop makes up.

  • Question 2: Which countries have seen the greatest increase in crop yield and farming efficiency since 1961? To determine the answer, I will plot the average change in crop yields and the amount of land used to produce a fixed amount of crops since 1961 for each country using a categorical plot of some kind.

  • Question 3: How much has increased fertilizer and tractor use changed farming efficiency? To investigate, I will visualize the data associated with the two tools and land use.

3.1 Data

Using data from from the TidyTuesday GitHub page, I will analyze the popularity and features of different hiking trails in Washington state. The data comes from the file hike_data.rds, which has 1,958 observations 8 variables. The variables in this data are name, the name of the trail as a character, location, the location of the trial as a character, length, the length of the trail as a character, gain, a character representing the gain in elevation as feet above sea level, highpoint, the max elevation of the trail as a character, rating, a character value representing a numeric user rating out of 5, features, a character list of features for each trial, and description, and character value giving a brief description of every trail.
